# Local Storage Agent

## จุดมุ่งหมาย

สร้างแอปพลิเคชัน To-Do List ที่จัดเก็บข้อมูลรายการที่ต้องทำบน Local Storage ของเบราว์เซอร์
ให้ใช้งานสะดวก เป็นมิตรกับผู้ใช้ ใช้ได้ทั้งภาษาไทยและอังกฤษ
สามารถนำโค้ดไป deploy ด้วย GitHub Pages ได้อย่างสะดวก
รองรับการเพิ่มรายการ, ทำเครื่องหมายเสร็จแล้ว, ลบ และจดจำข้อมูลไว้ใน Local Storage โดยไม่ต้องสมัครสมาชิก

## วิธีใช้

1. เพิ่มไฟล์ index.html ลง repository นี้
2. เปิดใช้งาน GitHub Pages (Settings > Pages > main/root)
3. เข้าผ่าน <https://gittisak-go.github.io/Local-Storage-Agent/>
